Review Questions:

How do we invoke a function?

  • Using parentheses after the name of the function.
  • Argument list go inside in the order defined in the parameter list

What does DRY stand for?

  • Don't Repeat Yourself

Difference between parameter and argument?

  • Argument is a specific value for a parameter
  • Parameters are named in the function declaration

What does the return keyword do?

  • Stops the function
  • Return a value:
    • Result of an expression
    • An explicit value
    • Return value of an invoked function in the return
    • Can also return void (nothing)

exits the function

  • Gives control back to the caller
